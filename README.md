# chat-room
A simple chat Room!

Technologies & Design
-
- NodeJS
- MongoDB
- FastifyJS
- MongooseJS
- VueJS
- Vue-MC
- Vue-Router
- Socket.io
- Material Design

# Requirements
- NodeJS
- MongoDB
- Linux System

# Steps
```bash
$git clone https://github.com/eduardobc88/chat-room.git
$cd chat-room
$npm install
$npm run pm2:start
```
Enter to http://localhost:3000

# Commands
```bash
$npm run pm2:global-logs
$npm run pm2:kill
$npm run pm2:start-watch
$npm run webpack:site-watch
```
